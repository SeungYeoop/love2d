# One Shot Prototype Prompt

먼저 `AI_WORKFLOW.md`, `PROJECT_RULES.md`, `TASK.md`, `STRUCTURE.md`, `SESSION_SUMMARY.md`를 읽고 시작해줘.

내 목표는 짧은 설명만으로도 만족도 높은 LÖVE2D 프로토타입을 빠르게 만드는 거야.

아래 순서로 진행해줘.

- 내가 요청한 게임을 3~6줄로 요약해줘.
- 막히지 않으면 합리적인 가정을 적고 바로 구현해줘.
- 첫 결과물은 구조가 있는 playable prototype 또는 vertical slice로 만들어줘.
- `main.lua`에 모든 걸 몰아넣지 말고 필요해지면 작은 모듈로 분리해줘.
- 조작법, 목표, 실패/재시작, 기본 피드백을 포함해줘.
- 작업이 끝나면 `.\play.cmd`를 실행하거나 watcher 기준으로 결과 반영 여부를 확인해줘.
- 마지막에는 다음 개선 3개를 추천해줘.
- 꼭 필요한 경우에만 질문을 1~3개 해줘.
