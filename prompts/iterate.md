# Iteration Prompt

먼저 `AI_WORKFLOW.md`, `PROJECT_RULES.md`, `TASK.md`, `STRUCTURE.md`, `SESSION_SUMMARY.md`를 읽고 시작해줘.

이번 라운드에서는 이미 있는 프로토타입을 더 만족스럽게 개선해줘.

아래 규칙을 지켜줘.

- 현재 상태를 2~4줄로 요약해줘.
- 이번 요청이 구조 변경인지, 기능 추가인지, 감각 개선인지 먼저 분류해줘.
- 기존 재미 요소는 가능하면 유지하고, 이번 요청에 직접 필요한 부분만 수정해줘.
- 영향이 크면 짧은 계획을 먼저 보여주고 승인받아줘.
- 코드가 커지면 작은 파일/모듈 단위로 분리해줘.
- 작업이 끝나면 `.\play.cmd`를 실행하거나 watcher 기준으로 결과 반영 여부를 확인해줘.
- 마지막에는 다음 개선 3개 또는 필요한 질문 1~3개를 남겨줘.
